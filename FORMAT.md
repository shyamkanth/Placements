# How to write `.md` file
>Basic syntax and formatting for a `.md` file, you should know.

# Heading 
`# Heading-name`

## Subheading
`## Subheading-name`

### Minor Heading
`### Minor-heading-name`

# Styling Text
`**Text**` | **Bold**

`*Text*` | *Italics*

`~~Text~~` | ~~Strikethorough~~

`<sub>Text</sub>` | <sub>Subscript</sub>

`<sup>Text</sup>` | <sup>Superscript</sup>

### Single line code

`#include<iostrem>`

### Multiple line code

>Use triple back-ticks to wrap the code both from up and down side. 
```
#include<iostream>

using namespace std;

int main(){
    cout<<"Hello world;
    return 0;
}
```
